Was sind If Statements

Die IF-Anweisung ist eine Entscheidungsanweisung, die ein Programm anleitet, Entscheidungen auf der Grundlage bestimmter Kriterien zu treffen. Die IF-Anweisung führt einen Satz von Code aus, wenn eine bestimmte Bedingung erfüllt ist (TRUE) oder ein anderer Satz von Code zu FALSE ausgewertet wird.
